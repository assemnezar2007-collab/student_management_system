# student_management_system

A simple C++ console-based Student Management System that allows users to manage students and their enrolled courses through an interactive menu.

Features
Add a new student
Delete an existing student
Add a course to a student
Remove a course from a student
Display all students and their courses
Exit the program
Menu
===== Student Management System =====

1. Add Student
2. Delete Student
3. Add Course to Student
4. Delete Course from Student
5. Display All Students
6. Exit
Project Structure

The system includes the following main functions:

addStudent();
deleteStudent();
addCourseToStudent();
deleteCourseFromStudent();
displayAllStudents();
menu();
How It Works
1. Add Student

Allows the user to enter student information and store it in the system.

2. Delete Student

Removes a student from the system using their ID or identifier.

3. Add Course to Student

Enrolls a student in a new course.

4. Delete Course from Student

Removes a specific course from a student's course list.

5. Display All Students

Shows all registered students along with their enrolled courses.

6. Exit

Terminates the application.
